

> 闪念便签/Note是一款开源的Android便签，实现了主流便签的基本功能，并不断完善中。
> 特点： Material Design设计、MVP设计模式；


## 注意：
1. 本程序代码只用于学习交流，不可商用；
2. 代码下载后，App的反馈模块需要使用Bmob，请申请AppId后，在res/values/strings中修改bmob_app_id的值。

## 关于软件：

基本实现了常用的便签功能。以及模板可以自定义,启动界面可以自定义功能，快快快！
